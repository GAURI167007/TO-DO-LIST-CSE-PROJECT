PROBLEM STATEMENT:
People often struggle to track daily tasks, leading to missed deadlines and lower productivity. Manual methods like notebooks are inefficient and error-prone. A solution that 
organizes, prioritizes, and motivates task completion helps manage workloads effectively and improve productivity.

SCOPE OF THE PROJECT:
The To-Do List project helps users manage their daily tasks efficiently. Users can add, edit, delete, and mark tasks as complete. It tracks 
deadlines and priorities to keep tasks organized. Tasks are saved locally for future access. The project demonstrates core Python concepts 
like file handling . It can be enhanced later with features like multi-user support,GUI and cloud synchronization.

TARGET USERS :
The unique and best part of this project is that it can be used by people of any age group as it will help everyone manage their time and be productive.
however it mainly targets school and college going students.

High Level Feature:
Add tasks to the list
View all tasks
Delete tasks from the list
Save tasks locally (using files like JSON)
Achievement Badges
